## 🏥 About This Project

This is the complete, production-ready static website for **Barala Hospital & Research Centre**, Chomu, Rajasthan. It is built with pure **HTML5, CSS3, and Vanilla JavaScript** — no build tools, no frameworks, no npm. Upload to any static host and it works immediately.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `barala-hospital-website`)
2. Upload all files **exactly as structured above**
3. Go to **Settings → Pages → Source: Deploy from branch → main / root**
4. Your site will be live at: `https://yourusername.github.io/barala-hospital-website/`

**No configuration required. No build step. Works immediately.**

## 📋 Compliance Notes

This website follows medical content compliance guidelines:

- No absolute medical success rate claims (e.g., no "100% success rate")
- Hospital-reported statistics attributed as "as reported by the hospital"
- No corporate registration identifiers (CIN) in public content
- Safe language used: "advanced multi-speciality healthcare", "experienced specialists"
- Emergency numbers clearly displayed; appointment WhatsApp kept separate
